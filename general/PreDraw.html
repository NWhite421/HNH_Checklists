A. Modelspace Requirements

The legal description matches between the property card, deed, and plat. The aerial appears to match the legal description.

The boundary, adjacent parcels, and roads are drafted. Each subject parcel segment has appropriate measurements (bearing and distance for lines, curve data for curves) (Reference). Easements are drafted and annotated. The subject parcel is identified with lot & block information.

Add linework for every easement shown on the plat, either through the typical diagram shown on the first page or as shown on the lot itself. Label each easement with its width and purpose. A typical width and description can be used if all sides are the same.

Add two (2) dimensions, one from each property corner, to the centerline of the right-of-way. Ensure the accuracy is showing two (2) decimal places.

Each road directly adjacent to the parcel has: the road name (if the road name is different from the plat, make the current road name large), plat name (if applicable), and a line containing the width of the right-of-way.
