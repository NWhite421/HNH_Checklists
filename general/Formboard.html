P-1. AutoCAD Setup
Formboards
Wooden boards installed to set limits on concrete pours. Concrete usually has not been poured yet. Asked for verification of building location or verify elevations.
Foundation
The building base made out of concrete. Asked for verification of elevations.
Update Point Groups

Open the point group "GROUND SHOTS", click on the "EXCLUDE" tab, "check the "INCLUDE POINT NUMBERS WITH" button, and click the "SELECT" button. The dialog will close and allow you to select cogo points. Select all the ground elevations shown during the boundary & topographic portion of the project. This will hide the elevations without deleting them.

If autoCAD warns there you are overriding points, you can select the Override option if the data is the same (i.e. control points being reinserted) or Use Next Available Number if you thing the data is not the same (i.e. same number, different shots).

Import Points

Import your field field (*.txt) into AutoCAD using the importpoints command with the PNEZD (comma-delinted) format.

Update Point Groups

Run the UPDATEALLPOINTGROUPS command to automatically apply styles to applicable points.

Change Modelspace Scale

You can easily change the scale of cogo points by changing the scale of the modelspace (lower right corner). The closer to 1" = 1', the smaller the display size

A. Modelspace

Hide the following list of layers (only if a site plan was completed):

CONCRETE-PROPOSED
UTILITIES-PROPOSED
TOPO-PROPOSED
TEXT-PROPOSED

For a list, see the online checklist.

Ensure that no information about property corners, boundary information, or adjacent parcel information has disappeared. If it has, turn the layer back on and adjust the object's layer to be correct.

Change the text inside the house from "PROPOSED X-STORY RESIDENCE" to "FORMBOARDS" or "FOUNDATION" as required. Change the FFE elevation to the highest elevation out of the "bldg" points.

Review the location of the house points compared to site plan location (if available). Perform one of the following three (3) actions:

Points are <0.2' from plan
You can leave the linework in the same location and only adjust the building dimensions and ties.
Points are >0.2' from plan or No site plan was completed
Draw the linework, building dimensions, and ties to the point locations.
Points are >2.0' from plan
Do not adjust any linework or dimensions. Notify a Surveyor of the observation immediately.

Change the text inside the house from "PROPOSED X-STORY RESIDENCE" to "FORMBOARDS" or "FOUNDATION" as required. Change the FFE elevation to the highest elevation out of the "bldg" points.

B. Paperspace

Set the text directly below the logo to say "FORMBOARD SURVEY" or "FOUNDATION SURVEY" as needed.

Palm Coast Only Under the title, add the City of Palm Coast permit number, found at this site.

If this survey was inherited at the formboard / foundation stage, ensure that Part 1. Pre-Draw and Part II(A). Boundary Survey items have been fully satisfied.
