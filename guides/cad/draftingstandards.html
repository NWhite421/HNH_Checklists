Drafting Standards

Text

In professional drafting, there are two factors when it comes to text. Font family and font size. For the font family, the majority of text is simplex. Other fonts that are used in specific scenarios include RomanC and calibri. For font sizes, we go off the Leroy Sizing Convention or the L-system. The L-system specifies the text size in decimal feet and is easily scalable based on your drawing's scale. The L-system works as such.

L-Number Paper Print Size L-Number Paper Print Size
L60 0.060' L125 0.125'
L80 0.080' L200 0.200'
L100 0.100' L240 0.240'
L120 0.120' L500 0.500'

AutoCAD Implementation

There are two ways to implement the L-system into AutoCAD; manually or using Annotative text. Both have their positives and negatives and should be considered for the project.

Manual Text Scaling

Manual scaling required that the CAD technician calculates the "real-world" scale of text based on the scale of the drawing they are annotating. To get the scaled size of text, you multiply the L-number paper print size by the scale your drawing is at. For instance, if I want to insert a text at L80 size in a 1" = 20' drawing, you would multiply 0.080 × 20 to get to a text size of 1.600'.

Paper Print Size × Drawing Scale = Text Size

[L120] 0.120' × [1" = 40'] 40' = 4.800'

Annotative Text Setting

AutoCAD has a feature where it will calculate the paper print size based on the active scale of the drawing, or the scale the drawing is currently set at. The text style itself has to be setup to allow this functionality. It does have additional functionality where it will hide the text if the viewport's scale or printing scale does not match the text's established scale. It also allows the same text object to be at different sizes in different locations with the content syncing between all scales.
