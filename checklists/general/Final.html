P-1. AutoCAD Setup
Update Point Groups

Open the point group "GROUND SHOTS", click on the "EXCLUDE" tab, "check the "INCLUDE POINT NUMBERS WITH" button, and click the "SELECT" button. The dialog will close and allow you to select cogo points. Select all the ground elevations shown during the boundary & topographic portion of the project. This will hide the elevations without deleting them.

Import Points

Import your field field (*.txt) into AutoCAD using the importpoints command with the PNEZD (comma-delinted) format.

Consideration If autoCAD warns there you are overriding points, you can select the Override option if the data is the same (i.e. control points being reinserted) or Use Next Available Number if you think the data is not the same (i.e. same number, different shots).

Update Point Groups

Run the UPDATEALLPOINTGROUPS command to automatically apply styles to applicable points.

Change Modelspace Scale

You can easily change the scale of cogo points by changing the scale of the modelspace (lower right corner). The closer to 1" = 1', the smaller the display size of your points.

A. Modelspace

Draft any new hardscape items such as driveways, sidewalks, an concrete pads.

Draft newly installed utilities, including the A/C pad, cleanouts, electrical meter, and septic tank cover.

Verify that the grades and ground elevations provide enough information showing proper water drainage.

Palm Coast Download the swale plan on the Palm Coast Permitting Site link. Verify that the swale is in conformance with the swale plan.

B. Paperspace

Set the text directly below the logo to say "FINAL SURVEY".

Palm Coast Under the title, add the City of Palm Coast permit number, found at this site.

If this survey was inherited at the formboard / foundation stage, ensure that Part 1. Pre-Draw and Part II(A). Boundary Survey items have been fully satisfied.

Add the final field data collection date to Surveyor's notes.

Surveyor's Note 2
Add onto the existing note with "; FINAL [DATE]"
Sample: DATE OF FIELD SURVEY: 01/01/2022; SITE PLAN 02/01/2022; FORMBOARD 03/01/2022; FINAL 04/01/2022;
