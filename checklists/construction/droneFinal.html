File Management
Link Project Folders

Create a directory link between the proejct folder on the main drive (Z:\) and the drone server (X:\Drone Projects\). Create a folder on the drone server with the project number. Place the folder from Pix4Dmapper inside the newly created folder.

Create a Pix4Dsurvey Project

Open Pix4Dsurvey and drag the .p4d file in the mapper folder onto the Pix4Dsurvey window to start the import process. The project name will be Job Number - Job Common Name (i.e. 19-06-037 - Infotainment). The project location will be X:\Drone Projects\Job Number (i.e. X:\Drone Projects\19-06-067). Click Apply on the "Define input coordinate reference system" window.
