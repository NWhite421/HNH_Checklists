Seminole County Rural Plat Checklist

Pre-Application
Application

A fully filled and signed application. The application must either be physically signed (all submittals) or be digittally signed and able to be verified by a third-party (digial submittal only).
Application Fee: $50.00 + processing fee
Application fee subtracted from subsequent plat application fee.

A narrative described the proposed development. Should include how access to the property will be given, any easements, and any potential variances.

A map showing the current conditions of the site and inforamtion about any important sections.

A sketch of the proposed changes to the land. Show existing parcel information and proposed linework and easements. Show and note any access easements to the property.
